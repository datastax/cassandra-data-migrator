# Get started with the Cassandra Data Migrator (CDM)

You can use Cassandra Data Migrator (CDM) for data migration and validation between databases based on Apache Cassandra:registered:.

> **IMPORTANT**
>
>To use CDM successfully, your origin and target clusters must be Cassandra-based databases with matching schemas.

CDM can be used for any size of migration. It is ideal for large or complex migrations that benefit from advanced features and configuration options. For a complete list of features, see the [CDM README](https://github.com/datastax/cassandra-data-migrator?tab=readme-ov-file#features).

## Install CDM

It is recommended that you always install the latest version of CDM to get the latest features, dependencies, and bug fixes.

For installation instructions, see the [CDM README](https://github.com/datastax/cassandra-data-migrator?tab=readme-ov-file#features).
